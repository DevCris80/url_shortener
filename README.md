# URL Shortener

Este proyecto es elaborado sin la IA, es un proyecto de aprendizaje para entender cómo funciona un acortador de URL. El proyecto está construido utilizando FastAPI, SQLAlchemy y una base de datos SQLite.

El fin del proyecto es enteder como se puede hacer un servicio escalable, por lo que luego se implementará una solución de cacheo utilizando Redis, y se implementará un sistema de autenticación para que los usuarios puedan gestionar sus URLs acortadas.
